# Анализ рейтингов больниц США | PostgreSQL, Python, статистика, PowerBI
## Цель - Выявление факторов, влияющих на оценку пациентов, и определение характеристик наиболее успешных больниц

### Стек технологий:
- СУБД: PostgreSQL (VIEW с LEFT JOIN)
- Язык запросов: SQL (агрегация, джойны, фильтрация)
- Анализ и визуализация: Power BI (+ DAX для мер)
- Предобработка данных: Python (Pandas, NumPy), Power Query
- Статистика и тесты: Python (SciPy, Pingouin, Shapiro-Wilk, корреляция)
- Визуализация (исследовательская): Python (Matplotlib, Seaborn: regplot, boxplot, histplot, Plotly.Express)
- Финальный дашборд: Power BI (карта, scatter plot, bar chart, таблицы)
- Управление: Git

### Ключевые инсайты
1. Существует слабая обратная корреляция между количеством коек в больнице и рейтингом: меньше коек - больше рейтинг
2. Наличие экстренной помощи снижает рейтинг на 0,8 балла в среднем
3. Врачебные больницы - лучшие по среднему рейтингу, на втором месте - некоммерческие, на третьем - государственные и частные
4. Висконсин (WI) - штат с самыми рейтинговыми больницами (при количестве больниц >20)
5. Специализированные небольшие больницы (≤50 коек) имеют более высокие рейтинги, чем крупные многопрофильные госпитали с полным спектром экстренных услуг, что отражает разную сложность пациентов и специфику их работы.

### Выводы
- Маленькие частные клиники без экстренной помощи получают высшие оценки. Крупные государственные больницы требуют улучшения сервиса. 
- Штаты с самыми низкими средними рейтингами (Нью-Йорк, Невада) нуждаются в модернизации лечебных учреждений.
