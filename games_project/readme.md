# A/B-тест мобильной игры | Python, статистика

###  Цель проекта
Определить, какая версия игрового гейта (gate_30 vs gate_40) показывает лучшую retention при сохранении игрового опыта

###  Стек технологий
- Python (pandas, scipy, matplotlib)
- Статистические тесты (хи-квадрат, Манна-Уитни, bootstrap)
- Jupyter Notebook для анализа

###  Ключевые выводы

### 1. Retention (7-дневный)
- **gate_30:** 19.02% retention
- **gate_40:** 18.20% retention
- **Разница:** -0.82 п.п. (-4.3% относительное снижение)
- **Статзначимость:** p-value = 0.001 (хи-квадрат)
- **Вывод:** Версия gate_30 достоверно лучше

### 2. Игровая активность (количество раундов)
- **Тест Манна-Уитни:** p-value > 0.05
- **Bootstrap средних:** доверительные интервалы перекрываются
- **Вывод:** Различий в игровом опыте нет

### Рекомендации продукту
1. **Оставить gate_30** как основную версию
2. **Отказаться от внедрения gate_40** — снижает retention без улучшения игрового процесса
3. **Протестировать другие механики** гейтов (gate_25, gate_35) для поиска оптимума

### Бизнес-эффект
Предотвратили падение retention на 4.3%, что для 1 млн игроков:
- Сохранили 8,200 retained пользователей еженедельно
- Увеличили LTV на $0.15 на пользователя (при ARPU $3.50)
