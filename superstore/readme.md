# Отчет продаж Superstore
## Бизнес-инсайты и рекомендации

### Проблемы
Падение выручки на 29,2%, падение прибыли на 12,5$ (месяц к месяцу)
Мебель/Столы: маржа -18% при скидках 26%

### Рекомендации
**Срочные (квартал):** 
- Снизить скидки на cтолы с 26% → 15%
- VIP-программа для клиентов R1F1 (20% выручки)

**Стратегические (год):**
- Экспансия в Калифорнию, Нью-Йорк, Техас
- Динамическое ценообразование на основе анализа скидок

### Ключевые метрики
- Средняя прибыль: 12,4%
- Топ сегмент: клиенты R1F1 = $198K выручки
- Самые прибыльные штаты: Калифорния и Нью-Йорк ($76к и $74к соответственно)
- Лучшая категория товаров: Копиры (37,2% прибыли)
